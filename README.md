# Lillian Ruff Pet Spa

Official website for Lillian Ruff Pet Spa - a professional pet grooming salon located in Ronkonkoma, NY.

## Website

**[https://www.lillianruffpetspa.com](https://www.lillianruffpetspa.com)**

## Features

- Professional dog grooming services
- Boarding & daycare
- Membership plans
- Online product shop with Square payments
- Online appointment booking
- Blog with pet care tips

## Tech Stack

- Astro
- Tailwind CSS
- Square Payments API
- Vercel Hosting

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

The following environment variables are required:

```
PUBLIC_SQUARE_APPLICATION_ID=
PUBLIC_SQUARE_LOCATION_ID=
SQUARE_ACCESS_TOKEN=
SQUARE_ENVIRONMENT=
```
